### How to Run

```
$ git clone https://github.com/barclayd/Rock-Paper-Scissors.git

cd Rock-Paper-Scissors

open index.html
```

### How to Play

Choose Rock, Paper or Scissors by clicking on the appropriate icon and battle against the computer
